# Task Manager

A simple full-stack Task Manager app built with React (Vite) + Node.js/Express.

## Setup & Run

### Backend
```bash
cd server
npm install
node index.js
# Runs on http://localhost:5000
```

### Frontend
```bash
cd client
npm install
npm run dev
# Runs on http://localhost:5173
```
