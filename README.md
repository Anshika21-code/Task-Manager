## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Storage: In-memory (no database required)
- Deploy: Vercel (frontend) + Render (backend)

## Setup & Run

### Backend
```bash
cd server
npm install
node index.js
# Runs on http://localhost:5000
```

### Frontend
```bash
cd client
npm install
npm run dev
# Runs on http://localhost:5173
```
